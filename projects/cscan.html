3-Axis Hydrophone Scanning System

Automated Acoustic Field Characterization System for HIFU Transducers

Overview

This project focuses on the development of a 3-axis motion-controlled system for precise acoustic field mapping of High-Intensity Focused Ultrasound (HIFU) transducers. The system integrates hardware control, data acquisition, and signal processing for accurate pressure and intensity estimation.

Problem Statement

Characterizing the acoustic field of a focused ultrasound transducer requires high spatial resolution, stable positioning, and synchronized data acquisition. Commercial systems are expensive and inflexible, motivating the development of a custom experimental platform.

System Architecture

  • 3-Axis Motion System (Stepper Motors + Lead Screw)
  • Arduino Mega + TB6560 Drivers
  • Oscilloscope (Keysight DSOX1202A)
  • MATLAB VISA Interface for Data Logging
  • Python for Data Processing & Visualization

Implementation

The system operates using a synchronized motion-control and measurement protocol. At each spatial point, the hydrophone records voltage signals which are converted into acoustic pressure values. Data is logged via MATLAB and processed using Python to generate intensity maps.

Results

The system successfully generated:

  • Axial intensity profiles
  • 2D focal plane maps
  • 3D acoustic intensity visualization

Key Learnings

  • Mechanical stability is critical for high-resolution scans
  • Reflections significantly affect measurement accuracy
  • Signal processing is essential for reliable intensity estimation
